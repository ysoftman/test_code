# react app test

## 앱 생성시

```bash
# React + Vite (CRA 는 deprecated 라 vite 사용)
bun create vite@latest react-sample --template react
```

## 앱 실행

```bash
bun run dev
```

## update package dependency

```bash
# (bun.lock)
bun update

# npm update (package-lock.json) 는 너무 느려서 사용하지 말자.
```

## 일괄 포맷팅

```bash
# biome.json 설정 참고
biome format --write
```
