# react app test

## 앱 생성

```bash
mkdir ysoftman-react-app
npx create-react-app ysoftman-react-app
```

## 앱 실행

```bash
cd ysoftman-react-app
npm start
# 또는
bun start
# 또는
yarn start
```

## update package dependency

```bash
# (bun.lockb)
bun update

# npm update (package-lock.json) 는 너무 느려서 사용하지 말자.
```
